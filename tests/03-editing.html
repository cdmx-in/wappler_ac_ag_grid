Cell editable — all fields

Double-click any cell to edit. cell_data_edited fires for each change with { field, oldValue, newValue, rowId }. Logged below.

Row editable — restricted fields

row_editable=true with editable_fields="salary,rating,active" — only those columns are editable. row_data_edited fires once per row commit.

Select editors

cstatic_select_editors for department, cdynamic_select_editors for active. Both render a dropdown editor when the cell is edited.
