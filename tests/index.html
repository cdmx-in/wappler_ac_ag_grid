How to run

Serve the project root over HTTP (e.g. npx http-server . -p 8080 -c-1) then visit /tests/index.html. Each scenario waits until document.body.dataset.gridsReady === 'true' before the grids are interactable — agents can poll that flag instead of guessing timings.

Console helpers: TestKit.grid(id), TestKit.waitForGrid(id), TestKit.snapshot(id), TestKit.callMethod(id, name, ...args). Version drift between the running AppConnect build and tests/VERSIONS.json is logged on every page.

Scenarios

Each scenario isolates a feature group so failures are easy to attribute.

01 · Basic Render core

Default column inference, sort, filter, floating filter, pagination, count binding.

02 · Themes & Locale visual

All five built-in themes, dark mode toggle, RTL, locale switch (EN/HE/RU/ES/PT).

03 · Editing edit

Cell editable, row editable, static/dynamic select editors, edited-event payloads.

04 · Selection & Row Events events

Single/multi-row selection, row checkboxes, row click / double-click / status events.

05 · Action Buttons actions

Built-in edit/view/delete + 5 custom action buttons with conditions and class toggles.

06 · Export io

CSV / XLSX / PDF download, trim, exclude hidden, exclude fields, strip HTML, custom filename.

07 · Import io

importFileData() with CSV and Excel sources, surfacing fileData binding.

08 · Formatting & Types format

Custom column types, amount fields with precision, date format + timezone, column align.

09 · Grouping & Aggregation group

group_config, columns_to_sum / columns_to_count, footer sum precision.

10 · State Persistence state

Save/reset column state to localStorage, applied filter introspection, programmatic filters.

11 · Layout & Scroll layout

autoHeight vs normal layout, fixed header/footer, horizontal scroll, hide_id_field.

12 · Component Methods api

reloadGrid, loadGrid, destroyGrid, pinColumns, hideColumns, quickFilter, applyFilters, clearFilters.

Version manifest

If the AppConnect pill in the header doesn't match, run every scenario and bump tests/VERSIONS.json.
